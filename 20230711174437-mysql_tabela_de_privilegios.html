Mysql: Tabela de privilégios

Tabela de privilégios:

Nome da coluna Privilégio Descrição
Selectpriv SELECT Usuário pode executar instruções SELECT
Insertpriv INSERT Usuário pode executar Instruções INSERT
Updatepriv UPDATE Usuário pode executar Instruções UPDATE
Deletepriv DELETE Usuário pode executar Instruções DELETE
Createpriv CREATE Usuário pode criar bancos de dados e tabelas
Droppriv DROP Usuário pode remover bancos de dados e tabelas
Reloadpriv FLUSH Usuário pode executar Instruções FLUSH ou comandos equivaletes a mysqladmin
Shutdownpriv SHUTDOWN Usuário ligar e desligar o servidor
Processpriv PROCESS Usuário pode mostrar informações sobre processos ativos (SHOW PROCESSLIST)
Fliepriv FILE Pode ler e gravar arquivos no servidor, também necessário para criar tabelas externas
Grantpriv GRANT Usuário pode conceder os todos os privilégios que ele mesmo possui
Referencespriv REFERENCES Usuário pode relacionar tabelas com uso de chaves extrangeiras
Indexpriv INDEX Usuário pode criar um indice em uma tabela
Alterpriv ALTER Usuário pode executar Instruções ALTER TABLE
Showdbpriv SHOW DATABASES Usuário pode listar bancos de dados. Sem esse privilégio o usuário só recebe retorno dos bancos em que possui privilégios em tabelas
Superpriv SUPER Usuário pode executar Instruções de superusuário
Createtmptablepriv CREATE TEMPORARY TABLES Usuário pode criar tabelas temporárias
Locktablespriv LOCK TABLES Usuário pode adquirir bloqueios explicitos
Execpriv EXECUTE Usuário pode executar procedimentos armazenados ou funções
Replslavepriv REPLICATION SLAVE Contas utilizadas em servidores escravos e mestres precisam desse privilégio. Serve para obter atualizações feitas no mestre
Replclientpriv REPLICATION CLIENT Usuário pode executar as instruções SHOW MASTER STATUS e SHOW SLAVE STATUS
Creatviewpriv CREATE VIEW Usuário pode criar uma exibição
Showviewpriv SHOW VIEW Usuário pode mostrar a instrução CREATE VIEW usando a instrução SHOW CREATE VIEW
Createroutinepriv CREATE ROUTINE Usuário pode criar progrmas armazenados usando as instruções CRETE PROCEDURE e CREATE FUNCTION
Alterroutinepriv ALTER ROTINE Usuário pode alterar caracteristicas de funções armazenadas usando a instrução ALTER FUNCTION
Createuserpriv CREATE USER Usuário pode criar um usuário
Eventpriv EVENT Usuário pode criar, alterar e descartar eventos
Triggerpriv TRIGGER Usuário pode executar triggers
Createtablespacepriv CREATE TABLESPACE Usupario pode criar TABLESPACES
