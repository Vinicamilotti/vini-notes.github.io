Mysql: Tabela de funções de criptografia

Funcões de criptografia

Função Descrição
AESDECRYPT() Descriptografa uando AES
AESENCRIPT() Criptografa usando AES
CROMPRESS() retorna o resultado como uma string binária
MD5() Calcula a soma de verificação MD5
RANDOMBYTES() Retorna um vetor de bytes aleatório
SHA1(),SHA() Calcula uma soma de verificações de SHA-1 de 160bits
SHA2() Calcula a soma de verificação SHA-2
STATEMENTDIGEST() Valorde hash de resumo de instrução de computação
STATMENTDIGESTTEXT() Computa resumo de instrução normalizado
UNCUMPRESS() Descompacta uma string compactada
UNCOMPRESSLENGHT() Retorna o comprimento de uma string antes da compressão
VALIDATEPASSWORDSTRENGTH() Determina a força de uma senha
